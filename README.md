Ce projet est une application full-stack utilisant Spring Boot pour le backend et Angular pour le frontend.

Prérequis

Général

Java 17 (JDK)

Node.js et npm (v16 ou supérieur recommandé)

MySQL (serveur de base de données)

Dépendances

Backend

Spring Boot 3

Spring Security

MySQL Driver

Frontend

Angular Core 14.2

Angular Material
